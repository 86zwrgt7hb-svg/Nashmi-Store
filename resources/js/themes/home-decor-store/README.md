# Home Décor & Furniture Store Theme

A premium theme designed specifically for home décor and furniture businesses, featuring warm color palettes and elegant design elements.

## Theme Features

### Design Elements
- **Color Scheme**: Warm amber and orange gradients with earth tones
- **Typography**: Serif fonts for elegance, sans-serif for readability
- **Layout**: Spacious, gallery-style product showcase
- **Icons**: Home décor specific icons (🛋️, 🛏️, 🪑, etc.)

### Components
- **Header**: Premium navigation with search and user menu
- **HeroSection**: Luxurious hero with gradient backgrounds and call-to-actions
- **CategoryTabs**: Horizontal scrolling tabs with home décor icons
- **ProductCard**: Elegant product cards with hover effects
- **Footer**: Comprehensive footer with newsletter signup

### Unique Features
- Category-specific icons for furniture types
- Gradient overlays and decorative elements
- Premium product card design with ratings
- Responsive design optimized for furniture catalogs
- Warm color palette perfect for home décor brands

## Usage

The theme is registered as `'home-decor'` in the theme factory and can be used by setting a store's theme to `'home-decor'`.

## Customization

All components use Tailwind CSS classes and can be easily customized by modifying the color schemes and layout structures while maintaining the same functionality as other themes.
