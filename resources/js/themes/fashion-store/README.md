# Fashion Store Theme

A modern, elegant e-commerce theme designed specifically for fashion and clothing stores.

## Features

- **Fashion-Focused Design** - Elegant layouts optimized for clothing and accessories
- **Visual-First Approach** - Large product images, lookbooks, and style galleries
- **Complete E-commerce System** - Full shopping cart, checkout, and order management
- **Mobile-first Design** - Responsive layout optimized for all devices
- **Fashion-Specific Features** - Size guides, color variants, style recommendations

## Components

- `FashionStore` - Main theme component
- `Header` - Fashion-focused header with search, wishlist, cart
- `HeroSection` - Large banner with fashion imagery
- `CategoryGrid` - Visual category navigation
- `ProductGrid` - Fashion product display with hover effects
- `Footer` - Elegant footer with brand information

## Styling

Fashion-focused color palette with elegant blacks, whites, and rose gold accents.
